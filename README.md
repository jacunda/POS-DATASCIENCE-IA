# POS-DATASCIENCE-IA

Projetos desenvolvidos no MBA FIAP - Data Science & IA

## 📚 Sobre

Este repositório reúne os projetos e trabalhos desenvolvidos durante o MBA em **Data Science & Inteligência Artificial** da [FIAP](https://www.fiap.com.br/).

O programa aborda as principais tecnologias e metodologias utilizadas no mercado de dados, incluindo:

- Análise e visualização de dados
- Machine Learning e Deep Learning
- Processamento de Linguagem Natural (NLP)
- Visão Computacional
- Engenharia de Dados e Big Data
- MLOps e Deploy de Modelos

## 📁 Estrutura do Repositório

```
POS-DATASCIENCE-IA/
├── fase01-visualizacao-dados/    # Análise exploratória e visualização
├── fase02-machine-learning/      # Modelos de Machine Learning
├── fase03-deep-learning/         # Redes Neurais e Deep Learning
├── fase04-nlp/                   # Processamento de Linguagem Natural
├── fase05-visao-computacional/   # Computer Vision
├── fase06-engenharia-dados/      # Pipelines e Engenharia de Dados
└── tech-challenges/              # Desafios técnicos e projetos finais
```

## 🛠️ Tecnologias Utilizadas

- **Python** (pandas, numpy, scikit-learn, TensorFlow, PyTorch)
- **SQL** e bancos de dados relacionais/NoSQL
- **Apache Spark** para Big Data
- **Jupyter Notebooks** para análise interativa
- **Docker** e **MLflow** para MLOps

## 🚀 Como Utilizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/jacunda/POS-DATASCIENCE-IA.git
   cd POS-DATASCIENCE-IA
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Navegue até o projeto desejado e abra o Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## 📋 Requisitos

- Python 3.8+
- Jupyter Notebook / JupyterLab
- pip ou conda
